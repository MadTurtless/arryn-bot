# Arryn Aid
Welcome to the Arryn Aid repo! This hosts the code used by the Discord bot in House Arryn.
## Features
**Reaction Roles:** Automated regional role assignment using message reactions.
**Planned:** Event logging, auditing
## Setup
1. Clone the repo.
2. Create a `.env` file in the root directory with the required variables: `DISCORD_TOKEN`, `REACTION_ROLES_CHANNEL_ID` and `REACTION_ROLES_MESSAGE_ID`.
3. Install dependencies: `pip install -r requirements.txt`
